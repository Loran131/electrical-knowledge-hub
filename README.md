## 📖 项目简介

本项目旨在用通俗易懂的方式向公众普及电气工程相关知识，破除日常生活中的电气谣言，增强安全用电意识。

### 六大科普主题
1. ⚡ 一度电的旅程 — 从发电厂到你家的插座
2. 🔌 手机快充的秘密 — USB-PD协议与GaN氮化镓技术
3. 🧠 单片机：家电的大脑 — 无处不在的微型控制器
4. 🏠 智能家电如何"智能" — IoT四层架构解析
5. 📡 电磁辐射的真相 — 用科学破除常见谣言
6. 🔋 锂电池安全使用指南 — 从原理到安全防护
